# Online Course Registration System using Python and Django

# Project Overview
The Online Course Registration System is a web-based application developed using Python and Django to simplify and automate the process of course enrollment and management. The system allows students to create accounts, browse available courses, register for courses, and manage their enrollments online. Administrators can efficiently manage course details, student records, and enrollment activities through an admin dashboard. The project reduces manual work, improves accuracy, and provides a faster and more organized registration process.

# Project Workflow
User registers and logs into the system.
Student views available courses.
Student selects and enrolls in preferred courses.
Enrollment details are stored in the database.
Admin manages courses and monitors registrations.

# Installation Steps
Install Python.
Install Django using:
 pip install django
Clone or download the project.
Navigate to the project folder.
Run migrations:
 python manage.py migrate
Start the server:
 python manage.py runserver
Open browser:
Plain text
 http://127.0.0.1:8000/
